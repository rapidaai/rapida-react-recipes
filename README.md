# rapida-react-sample

This repository contains React samples and recipes that demonstrate use cases using the Rapida platform.

### Rapida Platform
- Website: [https://www.rapida.ai/](https://www.rapida.ai/)
- Documentation: [https://doc.rapida.ai/introduction/overview](https://doc.rapida.ai/introduction/overview)

### NPM Package
To get started, install the official Rapida React package:
```bash
npm install @rapidaai/react
```

Explore examples in this repository to learn how to integrate and use Rapida efficiently in different scenarios.
